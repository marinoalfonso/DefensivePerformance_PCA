# Data

## Source

All data are extracted at runtime from [**fbref.com**](https://fbref.com) using the [`worldfootballR`](https://jaseziv.github.io/worldfootballR/) R package.

The relevant scraping calls in `report.Rmd` are:

```r
df_ITA <- fb_season_team_stats("ITA", "M", 2024, "1st", "defense")[1:20, ]
df_GER <- fb_season_team_stats("GER", "M", 2024, "1st", "defense")[1:18, ]
df_FRA <- fb_season_team_stats("FRA", "M", 2024, "1st", "defense")[1:18, ]
df_SPA <- fb_season_team_stats("ESP", "M", 2024, "1st", "defense")[1:20, ]
df_ENG <- fb_season_team_stats("ENG", "M", 2024, "1st", "defense")[1:20, ]
```

## Caching (recommended)

To avoid repeated web requests — which may be throttled by fbref.com — save the merged raw data locally after the first extraction:

```r
# Save
saveRDS(df, "data/raw_defensive_2024.rds")

# Reload in subsequent sessions
df <- readRDS("data/raw_defensive_2024.rds")
```

The `.rds` file is **excluded from version control** via `.gitignore` because:

1. The data belongs to fbref.com / StatsBomb and is not redistributable.
2. The file can always be regenerated by re-running the scraping step.
